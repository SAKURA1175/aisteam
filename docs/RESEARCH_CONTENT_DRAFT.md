# 2. 研究内容

## 2.1 研究对象

本研究以高校本科生为研究对象，重点关注其在网络交流场景中的语言行为特征、网络安全风险识别能力以及数字素养表现。研究将依托数字人交互系统与XR虚拟仿真场景，围绕大学生在社交平台、学习平台、校园社区等典型网络环境中的表达方式、互动习惯和风险应对行为展开分析，探究技术干预在规范网络语言、提升网络安全防范意识、优化数字行为习惯方面的实际作用。

## 2.2 研究框架

本研究构建“技术开发-行为干预-效果验证”三位一体的总体框架，形成从平台建设到教学实施、再到效果评估的完整研究闭环。

1. 技术层。基于AI大模型与XR技术，开发数字人导师动态交互模块，构建高仿真的网络语言规范训练场景和网络安全风险模拟场景，实现文本、语音、表情、动作等多模态交互支持。
2. 教育层。围绕“语言规范矫正+风险情境模拟”双轨路径，设计可嵌入语言类课程、数字素养课程和网络安全教育课程的教学活动，形成面向课堂教学、实践实训和自主训练的应用方案。
3. 验证层。综合采用量化分析与质性研究方法，对学生在干预前后的语言行为变化、风险识别能力提升和学习体验反馈进行系统评估，提炼可复制、可推广的高校数字素养培育模式。

## 2.3 研究目标

1. 构建多模态动态语料库。整合XR虚拟场景与AI自然语言处理技术，采集大学生在数字人交互过程中的文本、语音、视觉等多模态数据，建立面向网络语言规范与网络安全教育的动态语料库。
2. 研发数字素养训练系统。开发集数字人交互、风险情境模拟、过程记录与反馈评估于一体的训练平台，支撑大学生在沉浸式环境中完成网络语言规范训练与网络安全防范训练。
3. 形成课程融合与推广方案。将自主研发的数字素养训练系统有机融入高校课程体系，形成“技术平台+教学设计+评价机制”协同推进的教育创新实践方案，为高校相关课程改革提供参考。

# 3. 思路方法

## 3.1 基本思路

本研究坚持素养本位导向，整合AI与XR技术，构建“网络语言规范+网络安全防范”双维度培育体系。研究以场景化学习为核心，通过XR技术模拟高风险、强互动、贴近真实的网络交流环境，引入数字人开展实时引导、反馈与纠偏，解决传统课堂教学中“知识讲授多、情境体验少，认知理解强、实践迁移弱”的问题，推动大学生在真实感较强的训练过程中逐步形成规范表达意识、风险识别能力和安全应对能力。

## 3.2 研究方法

1. 跨学科研究法。融合XR技术、人工智能大模型、语言学、教育学与传播学等相关理论，为网络语言行为干预与数字素养培育提供综合性研究视角。
2. 虚实场景构建法。基于XR技术搭建高仿真网络实训环境，运用3D建模、交互设计与物理引擎构建虚拟网络情境，增强学生对网络语言规范和网络风险防范的具身认知。
3. 多模态数据分析法。围绕学生在训练过程中的文本表达、语音交互、行为轨迹和反馈结果开展数据采集与分析，为干预效果评价提供客观依据。
4. 实验验证法。采用前测-干预-后测的实验设计，对比学生参与网络实训前后的变化情况，通过量化指标与学习成果对比检验干预效果。
5. 质性研究法。通过访谈、问卷、学习反思和课堂观察等方式，分析学生对数字人交互、虚拟仿真训练和课程融入机制的真实感受，进一步完善研究结论。

# 4. 可直接使用的简版表述

本研究以高校本科生为对象，聚焦其网络语言行为与网络安全防范能力，依托数字人交互系统和XR虚拟仿真场景，探索技术干预对大学生网络行为规范化与数字素养提升的作用机制。研究构建“技术开发-行为干预-效果验证”三位一体框架，在技术层开发数字人导师与虚拟训练场景，在教育层设计“语言规范矫正+风险情境模拟”双轨教学方案，在验证层通过量化与质性研究评估干预成效。研究拟构建多模态动态语料库，研发数字素养训练系统，并形成可嵌入高校课程体系的推广方案。方法上综合采用跨学科研究法、虚实场景构建法、多模态数据分析法、实验验证法与质性研究法，以实现对大学生网络语言规范与网络安全能力的系统培育。
