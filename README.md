# aisteam

AI teaching marketplace chatbot with teacher-scoped memory, customizable rules, and user-specific knowledge bases.

## Vision

Build a multilingual AI tutor marketplace where each teacher has isolated memory, isolated rules, and isolated knowledge retrieval.

## Core features

- Teacher-scoped long-term memory powered by retrieval and structured memory.
- Independent memory spaces between different teachers.
- Teacher rules that can be configured and versioned.
- User-specific private knowledge bases.
- Multi-language conversation support.
- Web, iOS, Android, and mini program delivery.

## Planned stack

- Frontend: Next.js, Taro, React Native
- Backend: NestJS or FastAPI
- Data: PostgreSQL, pgvector, Redis
- AI: OpenAI Responses API, File Search, custom memory service
