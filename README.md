# aisteam

一个面向教学市场的 AI 会话机器人项目，支持老师级长期记忆隔离、可配置规则和用户专属知识库。

## 项目愿景

构建一个多语言 AI 教学市场，让每位老师拥有独立记忆、独立规则和独立知识检索能力。

## 核心能力

- 基于检索与结构化存储的老师级长期记忆
- 不同老师之间的记忆完全隔离
- 支持老师规则配置与版本管理
- 支持用户专属私有知识库
- 支持多语言对话
- 支持 Web、iOS、Android 和小程序

## 计划技术栈

- Frontend: Next.js, Taro, React Native
- Backend: NestJS or FastAPI
- Data: PostgreSQL, pgvector, Redis
- AI: OpenAI Responses API, File Search, custom memory service
